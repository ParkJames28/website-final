Bachelor of Science in
Information Technology

It is an undergraduate degree program that focuses on the study of computer science, technology, and
how they are applied to solve real-world problems. The program is designed to provide students with
the necessary skills and knowledge in areas such as programming, database management, networking,
system analysis, and web development.


  • Core Areas of Study Computer Programming: Learning various programming languages like Java, Python, C++, and others.
    Software Engineering: Understanding software development lifecycle, methodologies, and project management.
    Database Management: Working with databases, including creating, managing, and optimizing data storage systems.
    Networking: Studying computer networks, protocols, and network security to ensure data is transferred securely.
    Web Development: Learning how to design, develop, and maintain websites and web applications.
    System Analysis and Design: Understanding how to design IT systems and analyze their functionality.
    Cybersecurity: Learning how to protect information systems from cyber threats and vulnerabilities.
    IT Project Management: Managing and leading IT-related projects, including resource allocation, timelines, and deliverables.
  • Skills Developed
    - Technical Skills: Proficiency in coding, understanding networks, databases, and security measures.

    - Problem-Solving: Ability to analyze technical issues and provide solutions.

    - Critical Thinking: Making informed decisions about technology solutions.

    - Project Management: Managing IT projects from concept to completion.
  • Career Opportunities
    BSIT graduates can pursue careers in various fields, such as:

    - Software Developer

    - IT Consultant

    - Database Administrator

    - Network Administrator

    - Web Developer

    - System Analyst

    - Cybersecurity Specialist

    - IT Project Manager
